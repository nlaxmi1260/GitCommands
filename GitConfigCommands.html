Git Config Commands

Git can be configured at system level ,globally and locally. Following commands are used to configure git user globally for Mac

Command Description
$ git config --global user.name "nlaxmi1260" This command is used to configure git user name globally
$ git config --global user.email "yaram.nlaxmi@gmail.com" This command is used to configure git user email globally
$ ssh-keygen -t ed25519 -C "yaram.nlaxmi@gmail.com" This command is used to generate ssh private and public key. It will ask for name -lakshmi,and ask for passcode just enter enter .It will generate public and private keys
$ touch ~/.ssh/config This command will create config file under ~/.ssh folder
$ vi ~/.ssh/config This command will open config file in vi editor. copy below code in insert mode and save the file.
Host github.com
AddKeysToAgent yes
UseKeychain yes
IdentityFile ~/.ssh/lakshmi
$ pbcopy < ~/.ssh/lakshmi.pub This command will copy the public key into clipboard.
In browser login to github.com , goto settings -> ssh and gpg keys ->click on new sshkey -> paste publickey -> give name This activity will save the ssh publickey

Above steps if followed exactly nagaLakshmi github account should be configured with local mechine.
